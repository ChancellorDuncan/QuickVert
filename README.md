# QuickVert
Simple currency converter application for my Business Programming Course

## 📌 Overview
The Currency Converter Application is a desktop tool built with **C# and .NET Framework** that allows users to quickly and accurately convert currencies. It integrates with the **Open Exchange Rates API** to provide real-time exchange rate data.

This project is designed for:
- Freelancers and small business owners working with international clients
- Online shoppers estimating foreign prices
- Travelers planning trips
- Students and general users learning about exchange rates

---

## 🎯 Objectives
- Provide accurate currency conversions using a real-time API.
- Offer a simple, user-friendly interface for quick conversions.
- Support multiple international currencies.
- Reduce manual calculation errors.
- Demonstrate API integration in a desktop application.

---

## 🏗 System Architecture
The application follows a **three-layer architecture**:
1. **Presentation Layer (UI)** – Windows Forms interface for user input and displaying results.
2. **Application Logic Layer** – Handles validation, conversion logic, and history management.
3. **Data Layer (API Service)** – Connects to Open Exchange Rates API via HTTP requests and JSON responses.

---

## ⚙️ Technology Stack
- **Language:** C#
- **Framework:** .NET Framework
- **UI Technology:** Windows Forms (WinForms)
- **API Service:** Open Exchange Rates
- **Data Format:** JSON via HTTP requests

---

## 🔑 Core Features
- Currency input and selection
- Real-time exchange rate retrieval
- Automatic conversion and result display
- Conversion history tracking
- Error handling (invalid input, API failures, no internet)
